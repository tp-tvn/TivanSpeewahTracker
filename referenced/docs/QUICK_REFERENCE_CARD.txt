┌─ UNDERSTANDING BUDGET VS ACTUAL STATUS ────────────────────────────────────┐
│                                                                             │
│ 🟢 GREEN     Spending within 10% of budget → Keep monitoring             │
│ 🟡 YELLOW    Approaching limit (80-90%)    → Alert management            │
│ 🔴 RED       Over budget or late schedule  → Escalate immediately        │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ COMMON CALCULATIONS ──────────────────────────────────────────────────────┐
│                                                                             │
│ Actual Cost = Metres Drilled × Budget Rate ($/metre)                      │
│ Variance (%) = (Actual - Budget) ÷ Budget × 100                           │
│ Overspend Warning = When Variance > 10%                                   │
│ Cost/Metre = Total Actual Cost ÷ Total Metres Drilled                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ KEY DASHBOARD READINGS ──────────────────────────────────────────────────┐
│                                                                             │
│ Actual Metres      = How much has been drilled (should match Planned)     │
│ Actual Cost        = How much money spent (should not exceed Budget)      │
│ Budget             = Approved spending amount for this period             │
│ Variance %         = Difference from budget (negative = good)             │
│ Cost per Metre     = Efficiency metric (lower = more efficient)           │
│ Status             = Overall health (Green/Yellow/Red)                    │

┌─ WEEKLY MONITORING ROUTINE (15 minutes) ───────────────────────────────────┐
│                                                                             │
│ 1. Open Dashboard → Click "📊 Budget vs Actual"                           │
│ 2. Scan Status column for any 🟡 Yellow or 🔴 Red                       │
│ 3. If RED: Click to see details → Investigate the variance               │
│ 4. If YELLOW: Note it → Plan to check next week                          │
│ 5. Click "📅 Gantt Chart" to check schedule health                       │
│ 6. Look for actual bars starting after planned (late starts)             │
│ 7. Document any issues in your project notes                             │
│ 8. Report to stakeholders if needed                                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘

┌─ TIPS FOR SUCCESS ────────────────────────────────────────────────────────┐
│                                                                             │
│ 1. Import Gantt early — Even rough dates beat default dates              │
│ 2. Set accurate rates — Wrong rates = wrong cost tracking                 │
│ 3. Use consistent naming — "HOLE-001" everywhere, not "hole-001"         │
│ 4. Check dashboard weekly — Don't wait for problems to surface           │
│ 5. Export monthly reports — Keep historical snapshots for audit          │
│ 6. Document edits — Note any manual corrections you make                 │
│ 7. Use work groups — Tracks accountability by contractor                 │
│ 8. Assign all holes to purposes — No orphaned drilling data             │
