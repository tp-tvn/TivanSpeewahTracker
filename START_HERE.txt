═══════════════════════════════════════════════════════════════════════════════
                          ❓ COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════════

Q: How do I open the dashboard?
A: Double-click: 🚀_OPEN_DASHBOARD.bat

Q: How do I stop the dashboard?
A: Close the browser tab or press Ctrl+C in the terminal window.

Q: How do I import drill data?
A: 1) Place CSV files in data/tmp_import/
   2) Open the dashboard
   3) Use the Admin tab to import

Q: Where is my data stored?
A: All data is in: data/drill_costs.db (a SQLite database)

Q: How do I backup my data?
A: Copy data/drill_costs.db to a safe location.
   Automatic backups are created in data/

Q: I lost some data, can I recover it?
A: Copy one of the backup files (data/drill_costs.db.backup_*)
   and rename it to data/drill_costs.db

Q: Can I edit the dashboard?
A: Yes! The source code is in app/*.py files.
   Changes require restarting the dashboard.

Q: Does it work offline?
A: Yes! The dashboard is completely offline.
   No internet connection needed.
